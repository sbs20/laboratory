# Terminal image viewer
Arch: `sudo pacman -S libcaca`
Debian: `sudo apt-get install caca-utils`
Ref: http://superuser.com/a/378787

`cacaview myimage.jpg`

# 7zip
```
sudo pacman -S p7zip

# Create archive
7z a archive.7z *

# No compression
7z a -mx0 archive.7z *

# Extract
7z e archive.7z ./
```
