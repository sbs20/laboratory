# Git

```
# Clone a repo
git clone https://github.com/sbs20/pi-recipes.git

# Change dir and edit a file
cd pi-recipes
nano README.md

# See status
git status

# Stage the changes
git add . # or specific files

# Commit
git commit -m "Updated README"

# Sync
git push origin master
```
